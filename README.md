## Installation (manual)

* Open plugins directory  
* In the folder we execute
```shell 
git clone  https://github.com/Allekslar/oc-dragdropajax.git allekslar/dragdropajaxshopaholic
```

* Go to your project root and run: `php artisan october:up`.
