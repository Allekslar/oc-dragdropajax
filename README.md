## Installation (manual)

* Open plugins directory  
* Create a folder allekslar
```shell 
mkdir allekslar
```
* Go to it
```shell 
cd  allekslar
```
* In the folder we execute
```shell 
git clone  https://github.com/Allekslar/oc-dragdropajax.git dragdropajax
```

* Go to your project root and run: `php artisan october:up`.
